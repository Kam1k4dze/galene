# Galene Astro Client

This is a web client for the [Galene](https://galene.org/) videoconferencing server, built with [Astro](https://astro.build/) and [React](https://react.dev/).

> [!WARNING]
> This project was written by LLM for personal use. There are no guarantees regarding code quality, security, or the absence of bugs. Use at your own risk.

## Development

```sh
pnpm install
pnpm dev
```

## Build

```sh
pnpm build
```
